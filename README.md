# HelperCP

Помощник таможенного координатора для проверки комплектов документов: PO, Invoice,
Packing List и дополнительных файлов. Приложение извлекает данные через Gemini,
затем сверяет количества, цены, суммы и остатки по PO обычными правилами в коде.

## Запуск

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Как работает

1. Пользователь загружает пачку документов.
2. Код локально извлекает текст/таблицы из PDF, Excel, Word и TXT, если это возможно.
3. Gemini классифицирует документы и извлекает структурированный JSON, используя
   и визуальные файлы, и локально извлеченный текст.
4. Схема приводит ответ ИИ к стабильной структуре.
5. Python-код сверяет Invoice и Packing List против PO.
6. Приложение показывает таблицы проверки, извлеченные позиции, остатки и историю PO.
7. После ручного подтверждения отгрузку можно сохранить в историю PO.

История частичных отгрузок хранится локально в SQLite-базе `.data/helpercp.db`.

## Важные правила

- Invoice может закрывать только часть PO. В таком случае сумма invoice
  проверяется как `invoice quantity * unit price`, а не сравнивается с полной
  суммой PO.
- Даты нормализуются из форматов вроде `05.06.2026`, `2026-06-05`,
  `05/06/26`, `5 June 2026`.
- Номера документов сравниваются с учетом пробелов, дефисов, `№`, регистра и
  похожих вариантов записи.
- Числа с OCR-подозрительными символами вроде `I00`, `1O0` или `O.5`
  нормализуются для расчета, но такие строки помечаются как `Need manual review`.
- При низком качестве документа помощник не должен выдумывать значения:
  строка уходит на ручную проверку.
